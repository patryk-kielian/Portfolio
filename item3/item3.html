Flask blog

github repository

Back-endowa część blogu internetowego napisana w ramach kursu 100 Days of Code: The Complete Python Pro Bootcamp for 2023 z wykorzystaniem Pythona i framework'u Flask. Strona obsługuje tworzenie kont i logowanie. Dane obsługiwane są przez bazę danych SQLAlchemy, a hasła są hashowane (SHA256). Posty mogą być tworzone, edytowane i usuwane przez administratora, a wszyscy użytkownicy mogą pisać komentarze.

Użyte narzędzia:

Python Flask SQLAlchemy
